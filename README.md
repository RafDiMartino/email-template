# email-template

Initial email template testing
Using MJML framework

## Notes

### MJML

`npm install mjml`

`npm install -g mjml`

`export PATH="$PATH:./node_modules/.bin"`

Convert MJML to HTML 

`mjml [input] -o [output]`

Watch the MJML file

`mjml --watch index.mjml -o index.html`
